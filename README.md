# Around the World Travel App
Around the World Travel App is a full stack JavaScript application with an Express server and MongoDB database.

### Server

Install server dependencies:

```
cd server
npm install
```

Seed the database:

```
npm run seeds
```

Run express:

```
npm run server:dev
```

### Client

```
cd ../client
npm install
npm start
```
